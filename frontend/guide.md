# Frontend guide

This folder contains everything the browser needs: page templates, styling, images, fonts, and JavaScript.

## File and folder purposes

- `__init__.py` marks this folder as the frontend module.
- `templates/` contains the HTML pages and reusable page pieces rendered by Flask.
- `static/` contains browser assets such as CSS, JavaScript, images, fonts, and the favicon.
- `guide.md` is this plain-language guide to the frontend folder.
