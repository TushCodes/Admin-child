# Frontend JavaScript guide

This folder contains browser scripts that make public pages and admin screens interactive.

## File purposes

- `about-page.js` handles gallery navigation, contact feedback, and chat setup on the about page.
- `animations.js` reveals page sections smoothly as visitors scroll.
- `consignments.js` manages admin consignment table interactions, editing, POD previews, and save prompts.
- `forms.js` improves shared form behavior in the browser.
- `index-page.js` handles home page gallery, contact feedback, and visitor interactions.
- `index.js` starts the shared frontend features.
- `main.js` contains shared page behavior used across the site.
- `menu.js` controls mobile and responsive menu behavior.
- `newsletter.js` handles newsletter form behavior.
- `performance.js` applies small browser performance improvements such as lazy-loading images.
- `track-tooltip.js` shows helpful tracking tooltips.
- `track.js` supports the public shipment tracking page.
- `warehousing-page.js` handles warehousing page interactions and chat setup.
- `admin/` contains JavaScript helpers used by admin pages.
- `core/` contains reusable frontend building blocks.
- `features/` contains standalone browser features.
- `guide.md` is this plain-language guide to frontend JavaScript.
