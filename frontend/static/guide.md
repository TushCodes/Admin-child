# Static assets guide

This folder contains files that are sent directly to the browser.

## File and folder purposes

- `assets/` contains organized CSS and component/page styling.
- `css/` contains shared and legacy CSS files.
- `fonts/` contains Font Awesome font files used by icons.
- `images/` contains site images and page artwork.
- `js/` contains browser JavaScript.
- `favicon.ico` is the small browser tab icon.
- `guide.md` is this plain-language guide to static assets.
