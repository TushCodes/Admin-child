# Test support guide

This folder contains helper code used by more than one test.

## File purposes

- `__init__.py` marks this folder as the test support module.
- `admin_dashboard.py` contains browser helper steps for the admin dashboard tests.
- `app_factory.py` creates a configured test app.
- `server.py` starts a local server for browser tests.
- `guide.md` is this plain-language guide to test support helpers.
