# Scripts guide

This folder contains helper commands for developers who run or check the app locally.

## File purposes

- `bootstrap_local.sh` sets up local environment files, database services, dependencies, and seed data.
- `run_playwright_consignments.py` opens a browser and manually checks the admin consignments page.
- `run_playwright_test.py` opens a browser and manually checks the admin dashboard.
- `guide.md` is this plain-language guide to the scripts folder.
