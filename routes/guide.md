# Routes module guide

This folder connects web addresses to the Python functions that render pages or handle form submissions.

## File purposes

- `__init__.py` marks this folder as the routes module.
- `main.py` handles the home, about, and contact pages.
- `pages.py` renders service pages from matching templates.
- `track.py` handles shipment tracking searches and POD downloads.
- `admin/auth_routes.py` handles admin login and logout pages.
- `guide.md` is this plain-language guide to the routes module.
