# Utilities module guide

This folder contains shared helper code used by several parts of the app.

## File purposes

- `content_negotiation.py` decides when a request expects JSON instead of a normal web page.
- `errors.py` identifies known database errors that need special handling.
- `logging.py` adds request IDs to log messages so issues are easier to trace.
- `guide.md` is this plain-language guide to the utilities module.
